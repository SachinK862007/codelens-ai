STEP-BY-STEP COMMANDS TO RUN LOCALLY
================================================================================

TERMINAL 1 - BACKEND SETUP
--------------------------------------------------------------------------------
1. Navigate to backend:
   cd codelens-ai/backend

2. Create virtual environment:
   python -m venv venv

3. Activate virtual environment:
   Windows (bash):     source venv/Scripts/activate
   Windows (cmd):      venv\Scripts\activate.bat
   Windows (PS):       .\venv\Scripts\Activate.ps1
   Linux/Mac:          source venv/bin/activate

4. Upgrade pip:
   pip install --upgrade pip

5. Install dependencies:
   pip install -r requirements.txt

6. Create environment file:
   cp .env.examples .env

7. Edit .env file (use notepad or any editor):
   notepad .env
   
   Minimum required settings:
   SECRET_KEY=change-this-to-something-random-and-secure
   DATABASE_URL=sqlite:///./codelens.db
   ALLOWED_ORIGINS=["http://localhost:5173", "http://localhost:8000"]

8. Initialize database:
   python -c "from database.connection import Base, engine; Base.metadata.create_all(bind=engine)"

9. Start backend server:
   python app.py

   OR using uvicorn:
   uvicorn app:app --reload --host 0.0.0.0 --port 8000

Backend should now be running at: http://localhost:8000
Test it: http://localhost:8000/health


TERMINAL 2 - FRONTEND SETUP
--------------------------------------------------------------------------------
1. Navigate to frontend:
   cd codelens-ai/frontend

2. Install dependencies:
   npm install

   If errors occur, try:
   npm install --legacy-peer-deps

3. Start frontend server:
   npm run dev

Frontend should now be running at: http://localhost:5173

STOPPING THE APPLICATION
================================================================================
1. Stop Backend:  Press Ctrl+C in Terminal 1
2. Stop Frontend: Press Ctrl+C in Terminal 2
3. Deactivate venv: deactivate


TROUBLESHOOTING
================================================================================

Problem: Port 8000 already in use
Solution: Use different port
   uvicorn app:app --reload --port 8001

Problem: Port 5173 already in use
Solution: Use different port
   npm run dev -- --port 5174

Problem: PyAudio installation fails
Solution: Skip it (only needed for voice features)
   pip install fastapi uvicorn pydantic sqlalchemy alembic requests

Problem: CORS errors in browser
Solution: Check backend/.env file has correct ALLOWED_ORIGINS

Problem: Database errors
Solution: Delete and recreate database
   rm codelens.db
   python -c "from database.connection import Base, engine; Base.metadata.create_all(bind=engine)"

Problem: Module not found errors
Solution: Make sure virtual environment is activated
   source venv/Scripts/activate

ENVIRONMENT VARIABLES (backend/.env)
================================================================================
Required:
  SECRET_KEY                    - Random secret for JWT tokens
  DATABASE_URL                  - Database connection string
  ALLOWED_ORIGINS              - CORS allowed origins

Optional:
  CLAUDE_API_KEY               - For AI features
  OLLAMA_HOST                  - For local AI model
  REDIS_URL                    - For caching
  VOICE_ENABLED                - Enable voice features
