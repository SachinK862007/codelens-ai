# Icons Directory

This directory contains the application icons for different platforms:

- `32x32.png` - Standard icon size
- `128x128.png` - Larger icon size
- `128x128@2x.png` - Retina display icon
- `icon.icns` - macOS icon format
- `icon.ico` - Windows icon format

To generate these icons, you can use an online favicon generator or image editing software.
